Conversione DEC → BIN

Algoritmo delle divisioni successive

Parole chiave: DIV (quoziente intero) e MOD (resto)

Passaggi dell’algoritmo

  1. Dividi il numero per 2
  2. Annota il resto (0 oppure 1)
  3. Prendi il quoziente intero
  4. Ripeti finché il quoziente diventa 0
  5. Leggi i resti dal basso verso l’alto

Schema logico

n ← numero decimale

MENTRE n > 0 FAI
    resto ← n MOD 2
    stampa resto
    n ← n DIV 2
FINE

Il numero binario si ottiene leggendo i resti dal basso verso l’alto.
💡 Il resto può essere solo 0 o 1 perché stiamo dividendo per 2.
