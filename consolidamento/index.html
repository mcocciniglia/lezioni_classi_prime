Google Fogli – Prova di consolidamento

SOMMA • MEDIA • MIN • MAX • SE (anche annidati) • Formattazione condizionale • Grafici

Consegna

Tempo: 40–50 minuti
Strumenti: solo Google Fogli
Obiettivo: consolidare formule e strumenti base in modo ordinato e leggibile

Cosa devi consegnare

  • Un file Google Fogli chiamato: Verifica_Fogli_Cognome_Nome_Classe
  • Deve contenere 2 fogli:
    • Dati (tabella + formule)
    • Report (riepiloghi + grafici)
  • Le formule devono essere visibili e scritte in modo chiaro.

Regole

  • Non cancellare righe/colonne della tabella: compila solo ciò che è richiesto.
  • Evita calcoli “a mente”: usa sempre formule.
  • Se sbagli una formula, correggila: non lasciare celle vuote.

Struttura del lavoro

  1. Crea 2 fogli: Dati e Report
  2. Nel foglio Dati incolla la tabella in A1:H13
  3. Aggiungi colonne I–N e inserisci le formule richieste
  4. Nel foglio Report crea riepiloghi e grafici
Suggerimento: prima completa tutte le formule, poi fai formattazione condizionale e grafici.

Nota: “Lode” vale SI/NO. Le verifiche sono voti in decimi. Le assenze sono giorni.

Parte A – Calcoli per riga (colonne I–N)

A partire dalla colonna I, crea queste intestazioni:

  • I1: Totale
  • J1: Media
  • K1: Min
  • L1: Max
  • M1: Esito
  • N1: Bonus

Compila le formule dalla riga 2 alla riga 13:

  1. Totale: somma di Verifica1–Verifica3
  2. Media: media di Verifica1–Verifica3
  3. Min: minimo tra Verifica1–Verifica3
  4. Max: massimo tra Verifica1–Verifica3
  5. Esito (SE annidati):
    • Se Media ≥ 8 → “Ottimo”
    • Altrimenti se Media ≥ 6 → “Sufficiente”
    • Altrimenti → “Insufficiente”
  6. Bonus (SE annidati):
    • Se Lode = "SI" e Assenze = 0 → 2
    • Se Lode = "SI" e Assenze > 0 → 1
    • Se Lode = "NO" → 0

Parte B – Formattazione condizionale

  • In colonna J (Media):
    • evidenzia in modo chiaro i valori < 6
    • evidenzia in modo chiaro i valori ≥ 8
  • In colonna M (Esito):
    • “Insufficiente” deve risaltare
    • “Ottimo” deve risaltare

Foglio 2: “Report”

Parte C – Riepiloghi (usa formule)

  • Media generale delle Medie (colonna J del foglio Dati)
  • Media più alta e media più bassa
  • Assenze totali
  • Numero di “Insufficiente” (conta le celle in colonna M con quel testo)

Parte D – Grafici (2 grafici)

  1. Grafico a colonne: Media (J) per Studente (B)
  2. Grafico (a torta o a colonne): Assenze (G) per Classe (C)
