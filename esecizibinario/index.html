Esercizi — Conversione DEC ↔ BIN

10 esercizi: 6 progressivi (DEC → BIN) + 4 inversi (BIN → DEC).

Promemoria rapido
  • DEC → BIN: dividi per 2, prendi i resti, leggi i resti dal basso verso l’alto.
  • BIN → DEC: somma i pesi delle posizioni con bit = 1 (… 128, 64, 32, 16, 8, 4, 2, 1).

Parte A — Progressivi (DEC → BIN)

Converti i seguenti numeri in binario con divisioni successive.

  1. 6 (DEC) → (BIN) = ________ Soluzione: 110
  2. 13 (DEC) → (BIN) = ________ Soluzione: 1101
  3. 25 (DEC) → (BIN) = ________ Soluzione: 11001
  4. 42 (DEC) → (BIN) = ________ Soluzione: 101010
  5. 73 (DEC) → (BIN) = ________ Soluzione: 1001001
  6. 156 (DEC) → (BIN) = ________ Soluzione: 10011100

Parte B — Inversi (BIN → DEC)

Converti i seguenti numeri in decimale usando i pesi delle posizioni.

  1. 1011 (BIN) → (DEC) = ________ Soluzione: 11
  2. 110010 (BIN) → (DEC) = ________ Soluzione: 50
  3. 10010101 (BIN) → (DEC) = ________ Soluzione: 149
  4. 1110010 (BIN) → (DEC) = ________ Soluzione: 114
Gestione tempo (consiglio): fai prima gli esercizi 1–6. Se avanzi, completa 7–10.
Suggerimento: se vuoi verificare velocemente un risultato, puoi riconvertire il BIN ottenuto in DEC (controllo incrociato).
